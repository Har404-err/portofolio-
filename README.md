## 🚀 Technical Highlights

This project transcends a simple static site, implementing advanced architectural patterns:

*   **Modern UI Engine:** Built with **HeroUI v3** for premium, accessible, and high-quality components.
*   **Next-Gen Styling:** Leveraging **Tailwind CSS v4** for cutting-edge utility-first performance.
*   **Fluid Motion Systems:** Complex physics-based animations via **Framer Motion**, including shared element transitions and scroll-driven effects.
*   **Adaptive UX:** Custom cursor logic with interactive magnetism and touch-aware feedback systems.
*   **Dynamic Media:** Real-time YouTube-integrated music player with API-driven search and streaming capabilities.
*   **Performance Metrics:** Optimized for instant-load with Vite, reaching high scores in core web vitals.

## 🛠️ The Arsenal (Tech Stack)

### Frontend
- **Framework:** React 19 (Latest stable)
- **UI Library:** HeroUI v3 (formerly NextUI)
- **Styling:** Tailwind CSS v4 + PostCSS
- **Animations:** Motion (Framer Motion) & Lottie
- **Icons:** Lucide React & DevIcons

### Development & Tooling
- **Build Tool:** Vite
- **Language:** TypeScript (Strict Mode)
- **Package Manager:** NPM

## 📦 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- NPM or PNPM

### Installation
1.  **Clone & Navigate**
    ```bash
    git clone https://github.com/Har404-err/Portofolio.git
    cd Portofolio
    ```

2.  **Environment Setup**
    ```bash
    npm install
    ```

3.  **Launch Local Environment**
    ```bash
    npm run dev
    ```
