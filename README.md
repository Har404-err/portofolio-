## 🚀 Overview

This portfolio is not just a static site; it's a statement of **architectural integrity**. Built with speed and interactivity in mind, it features:

*   **Cyber-Tech Aesthetic:** Deep dark slate background (`#030712`) with vibrant electric cyan accents (`#00f0ff`), glowing hover states, and CRT scanline effects.
*   **Interactive UX:** Custom cursor logic that fluidly adapts on both Desktop and Mobile (Touch), complete with interactive magnetic glows.
*   **Vector Animations:** Integrated with Lottie for smooth, scalable, and high-quality vector animations.
*   **Performance First:** Powered by Vite and React for instant loading and minimal latency.
*   **Responsive Design:** Optimized flawlessly for all viewports, from mobile screens to 4K monitors.

## 🛠️ Tech Stack

### Interface (Frontend)
*   **React 19**: Modern UI rendering engine.
*   **Vite**: Next-generation frontend tooling for ultra-fast builds.
*   **Tailwind CSS**: Utility-first styling for the Cyber-Tech design system.
*   **Framer Motion**: Complex scroll animations, spring physics, and page transitions.
*   **Lottie React**: Dynamic, lightweight JSON-based vector animations.

### Core Logic
*   **Node.js**: Backend runtime & automation engine.
*   **TypeScript**: Ensuring robust, type-safe development across the stack.

## 📦 Installation & Setup

To run this portfolio locally on your machine (or Termux):

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Har404-err/Portofolio.git
    cd Portofolio
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    Access the site at `http://localhost:5173` (or the local Network IP provided in the terminal).
